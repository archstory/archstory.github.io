> 开启摄像头 10秒


```python
import time
import picamera

camera = picamera.PiCamera()
try:
    camera.start_preview()
    time.sleep(10)
    camera.stop_preview()
finally:
    camera.close()
```

> 调节亮度

```python
import time
import picamera
with picamera.PiCamera() as camera:
    camera.start_preview()
    try:
        for i in range(100):
            camera.brightness = i
            time.sleep(0.2)
    finally:
        camera.stop_preview()
```

> 录像_ 一段可控制大小的视频

```python
import picamera

with picamera.PiCamera() as camera:
    camera.resolution = (640, 480)    //视频尺寸
    camera.start_preview()
    camera.start_recording('foo.h264')  //文件名字
    camera.wait_recording(20)    //文件大小
    camera.stop_recording()
    camera.stop_preview()
```
